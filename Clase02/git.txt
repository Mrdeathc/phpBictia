COMANDO DE GIT

Inicializar el repositorio en nuestro pc

git init
-----------------------------------------------------------------------------

configuramos los permisos de usuario (credenciales) de nuestro repositorio

Iniciar con usuario de github

$ git config --global user.name "mrdeathc"

configuramos los permisos de usuario (credenciales) de nuestro repositorio

$ git config --global user.email "charlsmakotojojojo@gmail.com"

------------------------------------------------------------------------------

$ git status
